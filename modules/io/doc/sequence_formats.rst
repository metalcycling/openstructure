Supported Sequence File Formats
================================================================================

FASTA
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

*Recognized File Extensions*
  .fasta, .fna, .fas, .fa, .fsa
  
*Format Name*
  fasta

*Mode*
  Read/Write

ClustalW
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

*Recognized File Extensions*
  .aln
  
*Format Name*
  clustal

*Mode*
  Read-only

PIR
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

This is the format used by the
`Protein Information Resource (PIR) <https://proteininformationresource.org/>`_.

*Recognized File Extensions*
  .pir

*Format Name*
  pir

*Mode*
  Read/Write

Promod 2
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

*Recognized File Extensions*
  .ali
  
*Format Name*
  promod

*Mode*
  Read-only
